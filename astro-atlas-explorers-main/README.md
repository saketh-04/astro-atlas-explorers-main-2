## Overview

AstroAtlas is a comprehensive full-stack educational platform for exploring celestial objects including planets, stars, and galaxies. Features include interactive 3D visualization, advanced search, user and admin dashboards, and detailed analytics.

## Quick Start

```bash
npm install
npm run dev
```

Access at `http://localhost:8080`
